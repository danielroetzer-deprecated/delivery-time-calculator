Delivery Time Calculator

Calculating delivery times from given opening hours and a delivery time, with Moment.js

Description

Aim of this code is to produce usable delivery times for a delivery service. The generated delivery times are based on opening hours and delivery time from a javascript object. There were various opening hours picked, to simulate different possible opening hours from restaurants.


Opening hours are as following

1 = Monday, 2 = Tuesday, etc.
If a number is missing, this means, that the restaurant is closed on this day.

Change Current Day and Time

You can leave 1 input empty, so only Day or Time needs to be given. If inputs are empty, the current Time and/or Day of your computer will be used.

Time input needs to be 4 characters long and in 24h format. For example 1:30 AM equals 0130, 8:45 PM equals 2045
